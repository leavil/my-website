# Arnau Gonzalez Almirall - Portfolio

Este es el repositorio del portafolio profesional de Arnau Gonzalez Almirall, un Ingeniero especializado en **Machine Learning, Deep Learning y MLOps**. 

El proyecto fue migrado desde una arquitectura estática (HTML/CSS) a una SPA de alto rendimiento utilizando las últimas tecnologías del ecosistema moderno web.

## 🚀 Tecnologías (Stack)

- **Framework Core**: [Next.js](https://nextjs.org/) (App Router) y React.
- **Estilos y UI**: [Tailwind CSS v4](https://tailwindcss.com/) y componentes de [shadcn/ui](https://ui.shadcn.com/).
- **Animaciones y Efectos**:
  - [GSAP](https://gsap.com/) & ScrollTrigger para animaciones secuenciales complejas.
  - [Framer Motion](https://www.framer.com/motion/) para transiciones fluidas.
  - [tsParticles](https://particles.js.org/) para fondos interactivos.
- **Scroll**: [Lenis](https://lenis.darkroom.engineering/) para una experiencia de *smooth scrolling* inmersiva.
- **3D y Gráficos**: [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/) para visualizaciones en 3D y [Chart.js](https://www.chartjs.org/) para métricas interactivas.

## ⚙️ Cómo ejecutar en local

Para ejecutar el servidor de desarrollo, primero necesitas tener [Node.js](https://nodejs.org/) instalado. 

1. Clona el repositorio e instala las dependencias:

```bash
npm install
```

2. Arranca el servidor de desarrollo:

```bash
npm run dev
```

3. Abre [http://localhost:3000](http://localhost:3000) en tu navegador para ver el resultado.

## 📁 Estructura del Proyecto

- `src/app/` - Rutas principales de Next.js (App Router), configuración de layouts globales y estilos base (`globals.css`).
- `src/components/` - Componentes reutilizables de React.
  - `ui/` - Componentes base generados por shadcn/ui.
- `legacy/` - Contiene la versión original de la web en HTML/CSS plano como referencia.

## 🎨 Diseño y UX

El portafolio tiene un enfoque técnico y futurista, utilizando fuentes como **Space Grotesk** para un aspecto moderno y **JetBrains Mono** para detalles que evocan entornos de desarrollo de software.
